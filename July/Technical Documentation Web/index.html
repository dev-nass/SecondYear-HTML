Is JavaScript and Java the same?

JavaScript and Java are both incredibly fruitful Computer Programming Languages that are both utilized massively in the present and well-informed world. Even though they are comparative to one another because of the similarities they both possesses, fundamentally, both are different from one another.

JavaScript is an object-oriented, high-level, often just-in-time compiled programming language that transform and brought significant changes into the Web Development industry. It was firstly named as LiveScript in 1995. Since LiveScript back then showed a lot of similarties, including expression syntax, naming conventions and basic control-flow with the uprising and still developing Java it was then renamed as JavaScript.

On the other hand, Java is a statically typed language with a huge variety of varibles, has good memory management ensuring the runtime of every program, and a collection of ready to use robust libraries that handle various tasks.

Core Components of Java
  • Java Development Kit (JDK) it is a complete package that equips developers with necessary tools and resources to create, compile, and run Java-based applications or softwares.
  • Java Runtime Environment (JRE) solely used for the purpose of executing Java applications or softwares. It includes the Java Virtual Machine (JVM) for executing bytecode and the essential libraries that Java applications commonly rely on
  • Bytecode This is the representation of a Java source code after it gets compiled. This bytecode is not machine-specific, meaning it is not tied to a particular computer architecture. Instead, it is designed to be executed by a virtual machine, specifically the Java Virtual Machine (JVM).
  • Java Virtual Machine (JVM) this provides a virtual environment for Java Applications to run on device or operating system without modification. JVM is platform dependent, and it interprets the compiled Java bytecode and translates it into machine-specific code at a runtime.

Each varibale only expect to take one specific data type, just like in real-life, a container for salt and sugar should be separated.

Variables are very imporatant part of every programming languages, they are usually the one's being manipulated by the developer or programmer to accomplish the main goal of the whole program. You will learn more about variables and the different type of data types on the next section

Data types

Here's some widely used data type in Java

Boolean data type: bool myBoolVariable = false; Data type such as this are commonly use to hold true or false values.

Byte data type: byte myByteVariable = (byte)0; This one is commonly used for the purpose saving memory on large size arrays.

Short data type: short myShortVariable = (short)0; Used to store short sized integers, mainly for the purpose of saving memory.

Integer data type: int myIntVariable = 24; One of the mostly used data type in any programming language, as a beginner, you will be mostly using this.

Long data type: long myLongVariable = 24; A bigger version of the integer data type that can store larger integer values.

Float data type: float myFloatVariable = 24.123f; A data type that can hold decimal numbers or floating values

Double data type: double myDoubleVariable = 24.123456; Bigger version of the float data type.

Character data type: char myCharVariable = 'a'; Data type that can is used for holding single letter values.

String data type: String myStringVariable = "apple"; And a data type that holds a series of characters also known as string.

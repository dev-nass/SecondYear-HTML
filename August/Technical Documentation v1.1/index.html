Variables and Data type

As much as how useful containers are in real life, so it does in programming. Variables acts as a container that can hold different data types.

There are various data types in JavaScript, but before we deep dive into those lets first discussed the difference between variable declaration and initilization.

  • Declaration is the act of declaring a varible, meaning it only has the let or const keyword and the uniquely given identifier;
  • On the other hand, initialization is the process of declaring it but this time assigning a value into it.

JavaScript is a dynamically typed programming language, meaning you don't have to use specific keywords, unlike other languages, to declare what data type that variable should only store.

These, however, made the programming language more tricky on some instances, but if utilized corretly could lead to a more readable and yet functional code.

Nonethesless here's the different data types that JavaScript has to offer:

Undefined 
let varName;

When a variable is only declared, not initialized, it will automatically be an undefined data type

String 
let variableName = "Hello There";

A string on the other hand is diffeerent, a string is a series of characters that signifies that this variable holds letter or letters.

Integer 
let variableName = 12;

Integer holds integer numeric value, be it negative numbers, zero or positive ones.

Float 
let variableName = 1.24;

Floating or float for short specifically holds numbers that has decimal points.

Boolean 
let variableName = True;

Boolean specializes on holding only two value, either true or false.

Be notify that all data types mentioned above are all called Primitive Data Types. In a beginner sense, all you gotta do is to focus on getting familiarised on these data types first.

JavaScript Array

Arrays are also data types that doesn't belong to the primitive one's mentioned earlier.

Arrays are also the big daddy of the primitive data types as it can store not just one but multiple values inside it, however storing multiple data types on them is not advisable.

Here's how you initialized them:

    let variableName = [1, 2.3, true, "Hello"];

Declaring them is a bit diffeerent from normal one's, aren't they? Array values are supposed to be enclosed in a square brackets [ ], and each element, as we call it, should be seperated by a coma.

To access these elements we can use their index's. Indexies starts by 0, as derived from computers, proceeding to 1 and so on.

Here's how you do it:

For Loop and Variations

Loops can contain block of code that can repeatedly execute all the instructions pass into them.

For loop is one the most used loop in any programming languages. Here's the structure in creating this loop.

    for(declaration; condition; incrementation){
                
     // loop body 
    
              }

However unlike any other programming languages JavaScript has two more different variations of this both of which are used for iterating an array.

These are:

    For of loop 
    for(let i of array){
                    
     // loop body 
    
                  }
    For in loop 
    for(let i in array){
                    
     // loop body 
    
                  }

For of loop access each index of the array and return the elements of each.

For in loop does the same but return the actual index, not the element. But do reminded that the index being returned is of string data type and not integer.
